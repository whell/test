  5. URL to image, this domain, filename has no extension

  2. mailto link
  3. PDF document (filename has .pdf extension)
  4. img url to fbcdn (shoul.d be ignored if fbcdn.net is in ignored domains list)
  5. img src, fbcdn.net, same as above
